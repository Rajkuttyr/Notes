# Simple HTML5 Drag-and-Drop File Upload Interface

## Program

```html
<!DOCTYPE html>
<html>
<head>
    <title>Simple Drag and Drop Upload</title>
    <style>
        #dropArea {
            width: 300px;
            height: 150px;
            border: 2px dashed #000;
            text-align: center;
            line-height: 150px;
            margin: 50px auto;
        }
    </style>
</head>
<body>

<h2 align="center">Drag and Drop File Upload</h2>

<div id="dropArea">Drag files here</div>

<p id="output" align="center"></p>

<script>
    var dropArea = document.getElementById("dropArea");
    var output = document.getElementById("output");

    dropArea.addEventListener("dragover", function(event) {
        event.preventDefault();
    });

    dropArea.addEventListener("drop", function(event) {
        event.preventDefault();
        var files = event.dataTransfer.files;
        output.innerHTML = "Uploaded File: " + files[0].name;
    });
</script>

</body>
</html>
```

---

## 1. Purpose of the Program

This program creates a simple drag-and-drop interface that allows the user to upload a file by dragging it into a specified area on the webpage.

---

## 2. HTML Structure

### (a) Drag Area

```html
<div id="dropArea">Drag files here</div>
```

- A `<div>` element is used as the file drop area.
- It acts as the container where users drag and drop files.

### (b) Output Area

```html
<p id="output"></p>
```

- This paragraph is used to display the uploaded file name.

---

## 3. CSS Styling

```css
#dropArea {
    width: 300px;
    height: 150px;
    border: 2px dashed #000;
    text-align: center;
    line-height: 150px;
}
```

- `width` and `height` define the size of the drop box.
- `border: dashed` gives a visual indication that it is a drop zone.
- `text-align` and `line-height` center the text inside the box.

---

## 4. JavaScript Working

### (a) Prevent Default Behavior

```javascript
dropArea.addEventListener("dragover", function(event) {
    event.preventDefault();
});
```

- By default, the browser opens the file when dropped.
- `event.preventDefault()` stops this default behavior.
- This allows us to handle the drop manually.

---

### (b) Handling File Drop

```javascript
dropArea.addEventListener("drop", function(event) {
    event.preventDefault();
    var files = event.dataTransfer.files;
    output.innerHTML = "Uploaded File: " + files[0].name;
});
```

- The `drop` event is triggered when a file is dropped.
- `event.dataTransfer.files` stores the dropped files.
- `files[0].name` gets the name of the first file.
- The file name is displayed on the webpage.

---

## 5. Events Used

- `dragover` → Detects when file is dragged over the drop area.
- `drop` → Detects when file is dropped.

---

## 6. Advantages of Drag-and-Drop Interface

- User-friendly
- Simple and modern design
- Improves user experience
- Easy file handling

---

## Conclusion

This program demonstrates how HTML5 drag-and-drop API works by:

- Creating a drop zone using a `<div>`
- Preventing default browser behavior
- Capturing dropped files using JavaScript
- Displaying file details on the webpage

Thus, it provides a simple custom file upload interface.
