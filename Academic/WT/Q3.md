# HTML Registration Page with JavaScript Validation

## Program: Customer Registration Form (White Board Manufacturing Company)

```html
<!DOCTYPE html>
<html>
<head>
    <title>Customer Registration</title>
    <script>
        function validateForm() 
        {
            var name = document.getElementById("name").value;
            var phone = document.getElementById("phone").value;
            var email = document.getElementById("email").value;

            var namePattern = /^[A-Za-z ]+$/;
            var phonePattern = /^[0-9]{10}$/;
            var emailPattern = /^[^ ]+@[^ ]+\.[a-z]{2,3}$/;

            if (!namePattern.test(name)) 
            {
                alert("Name should contain only alphabets.");
                return false;
            }

            if (!phonePattern.test(phone)) 
            {
                alert("Phone number must contain exactly 10 digits.");
                return false;
            }

            if (!emailPattern.test(email)) 
            {
                alert("Enter a valid Email ID.");
                return false;
            }

            alert("Successfully registered your details");
            return true;
        }

        function clearForm()
        {
            document.getElementById("regForm").reset();
        }
    </script>
</head>

<body>

<h2>White Board Manufacturing Company - Customer Registration</h2>

<form id="regForm" onsubmit="return validateForm()">

    <!-- Text Boxes -->
    Customer Name: <input type="text" id="name"><br><br>
    Phone Number: <input type="text" id="phone"><br><br>
    E-mail ID: <input type="text" id="email"><br><br>

    <!-- Gender Options -->
    Gender:
    <input type="radio" name="gender" value="Male"> Male
    <input type="radio" name="gender" value="Female"> Female
    <br><br>

    <!-- Combo Box -->
    Years of Experience:
    <select>
        <option value="<5">Less than 5</option>
        <option value="5-10">5 to 10</option>
        <option value="10-15">10 to 15</option>
    </select>
    <br><br>

    <!-- Checkbox -->
    <input type="checkbox"> Certificate Proof Attached
    <br><br>

    <!-- Command Buttons -->
    <input type="submit" value="Submit">
    <input type="button" value="Clear" onclick="clearForm()">

</form>

</body>
</html>
```

---

# Explanation

## 1. Form Fields Used

- **Text Boxes** → Customer Name, Phone Number, Email ID  
- **Radio Buttons** → Male, Female  
- **Combo Box (Select)** → Years of Experience  
- **Checkbox** → Certificate Proof Attached  
- **Buttons** → Submit and Clear  

---

## 2. Validation Rules Implemented

### Name Validation
- Allows only alphabets and spaces.
- Uses Regular Expression:
  ```
  /^[A-Za-z ]+$/
  ```

### Phone Number Validation
- Must contain exactly 10 digits.
- Regular Expression:
  ```
  /^[0-9]{10}$/
  ```

### Email Validation
- Checks proper email format.
- Regular Expression:
  ```
  /^[^ ]+@[^ ]+\.[a-z]{2,3}$/
  ```

---

## 3. Working of Program

- When user clicks **Submit**, `validateForm()` function is called.
- If any field is invalid:
  - Error message is displayed using `alert()`.
  - Form submission stops.
- If all fields are valid:
  - Message displayed:  
    **“Successfully registered your details”**
- Clear button resets the form using `reset()` method.

---

# Conclusion

This program demonstrates:

- Creating a registration form using HTML.
- Using various form controls like textbox, radio button, select, checkbox.
- Performing form validation using JavaScript.
- Displaying appropriate success and error messages.

Thus, the form ensures proper input validation before submission.
