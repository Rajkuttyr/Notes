# JavaScript Program to Create Popup Box, Alert Box and Confirm Box

## Program

```html
<!DOCTYPE html>
<html>
<head>
    <title>Popup Boxes Example</title>
    <script>
        function showAlert()
        {
            alert("This is an Alert Box!");
        }

        function showConfirm()
        {
            var result = confirm("Do you want to continue?");
            
            if(result == true)
            {
                alert("You pressed OK!");
            }
            else
            {
                alert("You pressed Cancel!");
            }
        }

        function showPrompt()
        {
            var name = prompt("Enter your name:");
            
            if(name != null && name != "")
            {
                alert("Hello " + name);
            }
            else
            {
                alert("No name entered.");
            }
        }
    </script>
</head>

<body>

<h2>JavaScript Popup Boxes</h2>

<button onclick="showAlert()">Alert Box</button>
<button onclick="showConfirm()">Confirm Box</button>
<button onclick="showPrompt()">Prompt Box</button>

</body>
</html>
```

---

# Explanation

## 1. Alert Box

```javascript
alert("This is an Alert Box!");
```

- Displays a simple popup message.
- Contains only an **OK** button.
- Used to show warning or information.

---

## 2. Confirm Box

```javascript
confirm("Do you want to continue?");
```

- Displays a popup with **OK** and **Cancel** buttons.
- Returns:
  - `true` if OK is pressed.
  - `false` if Cancel is pressed.
- Used for confirmation actions.

---

## 3. Prompt Box

```javascript
prompt("Enter your name:");
```

- Displays a popup with a text input field.
- Allows user to enter data.
- Returns the entered value.

---

# Conclusion

This program demonstrates three types of JavaScript popup boxes:

1. **Alert Box** – Displays a message.
2. **Confirm Box** – Asks for confirmation.
3. **Prompt Box** – Accepts user input.

These popup boxes are used for interaction between user and webpage.
