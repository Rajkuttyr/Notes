# CSS Program to Add Background Image and Text Indentation

## HTML Code

```html
<!DOCTYPE html>
<html>
<head>
    <title>Background and Indentation Example</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <div class="container">
        <h1>My Web Page</h1>
        <p>
            This paragraph demonstrates the use of background images and text 
            indentation using CSS. The first line of this paragraph is indented.
        </p>
    </div>

</body>
</html>
```

---

## CSS Code (style.css)

```css
body {
    background-image: url("background.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    background-position: center;
}

.container {
    width: 60%;
    margin: 100px auto;
    background-color: white;
    padding: 20px;
}

p {
    text-indent: 40px;
}
```

---

## Explanation (For Exam)

### 1. Background Image
- The `background-image` property is used to set an image as the background of a webpage.
- Example:
  ```css
  background-image: url("background.jpg");
  ```
- `background-size: cover;` makes the image cover the full page.
- `background-repeat: no-repeat;` prevents the image from repeating.
- `background-position: center;` centers the image.

---

### 2. Text Indentation
- The `text-indent` property is used to indent the first line of a paragraph.
- Example:
  ```css
  text-indent: 40px;
  ```
- This moves the first line 40 pixels to the right.

---

### 3. Container Styling
- `width` sets the size of the content box.
- `margin: auto;` centers the box horizontally.
- `padding` provides space inside the box.
- `background-color` improves text readability.

---

## Conclusion
This program demonstrates how CSS can be used to:
1. Add a background image to a webpage.
2. Control background display properties.
3. Indent the first line of text using `text-indent`.

These properties improve the appearance and readability of a webpage.
