# <font color="red">ArrayList</font>

**Syntax**

`ArrayList<Type> list = new ArrayList<>(); List<Type> list = new ArrayList<>();`

# Important Methods

1. `list.add(element)`
    
2. `list.add(index, element)`
    
3. `list.get(index)`
    
4. `list.set(index, element)`
    
5. `list.remove(index)`
    
6. `list.remove(Object o)`
    
7. `list.size()`
    
8. `list.contains(element)`
    
9. `list.isEmpty()`
    
10. `list.clear()`
    

## Use Case

- Dynamic array
    
- Random access (index-based)
    
- Store duplicates
    
- Maintain insertion order
    
- Used in sliding window problems
    

> [!TIP]  
> Best choice when **read operations are more than write operations**
